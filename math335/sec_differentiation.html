Section 4 Differentiation

It's time for calculus.

Definition 4.1.

Let \(f\) be a function, and \(x\) be a point in the domain of \(f\text{.}\) We say that \(f\) has derivative \(D\) at the point \(x\) if the following two conditions hold:

  • The point \(x\) is an accumulation point of the domain of \(f\text{.}\)

  • If \(S\) is an open interval containing \(D\text{,}\) then there is an open interval \(T\) containing \(x\) such that if \(t\in T\text{,}\) \(t\neq x\text{,}\) and \(t\) is in the domain of \(f\text{,}\) then

    \begin{equation*} \frac{f(t)-f(x)}{t-x}\in S\,\text{.} \end{equation*}

If \(f\) does indeed have a derivative at some points in its domain, we call the derivative of\(f\text{,}\) denoted \(f'\text{,}\) the function giving by sending such an \(x\) to the derivative \(D\) at the point \(x\text{.}\) We write \(f'(x)=D\) in that case. We also say that \(f\) is differentiable at such points.

Note that the definition of derivative automatically excludes the kind of behavior we saw with continuous functions, where a function defined only at a single point was continuous.

Example 4.2.

A function defined only on \(\mathbb{Z}\) cannot have a derivative.

In doing these, many people found the following observation useful.

  • Julie's Observation (JO3):.

    If a function's domain is \(\mathbb{R}\text{,}\) then every point is an accumulation point of the domain.

If \(f\) is differentiable at \(x\) and \(c\in\mathbb{R}\text{,}\) show that the function \(cf\) also has a derivative at \(x\) and \((cf)'(x) = cf'(x)\text{.}\)

Problem 4.8.

If \(f\) and \(g\) are differentiable at \(x\text{,}\) show that the function \(f+g\) also has a derivative at \(x\) and \((f+g)'(x) = f'(x)+g'(x)\text{.}\)

Some people found this useful; how would you prove the converse?

  • Shinae's Lemma:.

    If \(a\lt x\lt b\) and \(c\lt y\lt d\) then \(a+c\lt x+y\lt b+d\text{.}\)

When it comes to products, things are more subtle, as shown in stating the familiar product rule. On a different note, our facts about products from previous chapters can be used to show that differentiable functions are continuous. The following examples show the converse of this fact isn't true, and that even when \(f'\) does make sense, the domain of \(f'\) might be a proper subset of the domain of \(f\text{.}\)

Problem 4.9.

Let \(f(x)=|x|\text{.}\) Prove that \(f\) is differentiable everywhere except at \(x=0\text{.}\)

Problem 4.10.

Let \(f(x)=x\) if \(x\in \mathbb{Q}\text{,}\) and \(f(x)=0\) otherwise. Show that \(f\) is continuous at \(x=0\text{,}\) but not differentiable.

Proof.

Continuous follows from surrounding \(x=0\) and everybody getting small, but differentiable doesn't because the difference quotient is always either 1 or 0, depending on whether \(t\) is rational or not.

It's useful to spend just a little time thinking about how one would prove the usual facts about derivatives used in a calculus course.

Problem 4.12.

Suppose a function \(f\) defined on the closed interval \([a,b]\) has a derivative at some \(x\in(a,b)\) and \(f(x)\geq f(t)\) for all \(t\in(a,b)\text{.}\) Show that then \(f'(x)=0\text{.}\)

So if there is an extreme value of a differentiable function, it occurs at a critical point. This leads us to (a variant of) Rolle's Theorem.

Problem 4.13.

Suppose \(a\lt b\text{,}\) that \(f\) is differentiable on \((a,b)\text{,}\) continuous on \([a,b]\text{,}\) and that \(f(a)=f(b)=0\text{.}\) Show there exists a point \(c\in (a,b)\) such that \(f'(c)=0\text{.}\)

Now consider a more general function where \(f(a)\neq f(b)\text{.}\) If one carefully considers the secant line connecting the endpoints of the graph of this function and applies the previous problem to this, one can prove the following crucial result, the Mean Value Theorem:

Problem 4.14.

Suppose \(f\) is a continuous function on \([a,b]\) and differentiable on \((a,b)\text{.}\) Then there is a point \(c\in (a,b)\) such that

\begin{equation*} f'(c)=\frac{f(b)-f(a)}{b-a}\;\text{.} \end{equation*}

Among many other wonderful things this theorem provides, we have the following standard result.

Problem 4.15.

If the derivative of a function is uniformly zero on a closed interval, then the function is constant on that interval.
