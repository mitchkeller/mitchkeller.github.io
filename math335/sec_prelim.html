Section 1 Preliminaries and Review

In these notes, we do assume certain background knowledge. This section is intended to review some of this, as well as to focus on specific items which may prove of use later.

Subsection 1.1 Absolute Basics

There are some things I will not expand on at all.

  • We will take most normal mathematical and logical terminology from a course such as MATH 210 Transition to Abstract Mathematics for granted.

  • We assume that sets exist. We will take the sets of counting (\(\mathbb{N}\)), integer (\(\mathbb{Z}\)), rational (\(\mathbb{Q}\)), and real (\(\mathbb{R}\)) numbers as known. In this course \(\mathbb{N}\) starts with \(1\text{,}\) not \(0\text{.}\) We will denote the irrational numbers by \(\mathbb{Q}^{C}\text{,}\) a notation for the complement of the rationals with respect to the set \(\mathbb{R}\text{.}\)

  • In these sets we will assume the most basic properties of addition, subtraction, and multiplication, such as commutativity and distributivity. Likewise, we will later assume without proof basic facts about ordering numbers. At this time we explicitly assume that \(0\lt 1\) and that there are no integers between them. You will be able to use these facts to prove that is true for other consecutive pairs of integers. (The fact \(0\lt 1\)Â does require proof! You may recall mention of the well-ordering principle from MATH 210, which his part of this.)

  • We will assume the real numbers exist. (We will say more about their properties in a bit, though.)

  • Although there is a lot of fun to be had with infinity, we do not foresee doing much with it other than using generic and specific infinite sets.

The rest of this preliminary section is intended to remind us of things for which some review could be useful.

Subsection 1.2 Logic Review

In basic proof logic, we will usually be proving a statement of the nature

\begin{equation*} \text{ If } P\text{ is true, then } Q\text{ is also true }\text{.} \end{equation*}
Exercise 1.1.

Prove that if \(x\) is an even integer, then \(x^2\) is also an even integer.

However, there are more interesting examples of how to prove things. Recall the converse of the generic statement

\begin{equation*} \text{ If } Q\text{ is true, then } P\text{ is also true } \end{equation*}

and the contrapositive thereof

\begin{equation*} \text{ If } Q\text{ is not true, then } P\text{ is also not true }\text{.} \end{equation*}
Exercise 1.2.

State the converse of the previous example, and then its contrapositive. Prove that final statement.

In (normal) mathematical logic, we say that the contrapositive and the original statement are logically equivalent. So you should use whichever one you find useful in proving theorems!

However, that isn't the case with the converse and the original statement.

Exercise 1.3.

Take the statement âIf \(n\) is a prime number, then \(n\) is a positive integerâ and give a reason why it is true, but its converse is false.

In the first two examples, we have the highly unusual situation where both \(P\implies Q\) and \(Q\implies P\) are both true. We say that this is a biconditional statement and write \(P\iff Q\text{,}\) saying

\begin{equation*} P\text{ is true if and only if } Q\text{ is true }\text{,} \end{equation*}

or â\(P\) is necessary and sufficient for \(Q\)â, or some variation thereon.

There are a few other tricky logic issues that we should definitely practice. First, or is a word that means many things. In most of math (and here), âA or Bâ means âA or B or bothâ. This is important in statements like

\begin{equation*} \text{ If } x\text{ or } y\text{ is even, then } xy\text{ is also even } \end{equation*}

since even if \(x\) and \(y\) were both even, \(xy\) would still be even.

The other remaining tricky issue is that of quantifiers â things like âfor allâ and âthere existsâ. This is best tackled by trying it.

Exercise 1.4.

Restate these as accurately as possible in terms of logic, and then do all of the following with as few double negatives as possible, using natural negations if possible:

(a)

Negate the statement, âAll Morningside professors who live on the Northside are old and grey.â

(b)

Negate the statement, âThere exists a Morningside student who does not live in Sioux City,â in the most natural way possible (i.e., without double negatives).

(c)

Give the contrapositive of the statement, âIf there exists a Morningside professor who is neither old nor grey, then all Morningside students are young and bald.â

Exercise 1.5.

Formally negate the statement, âFor every positive number \(p\text{,}\) there is a positive integer \(N\) such that if an integer \(I\) is greater than \(N\text{,}\) then the reciprocal of \(I\) is less than \(p\text{,}\)â with as few negatives and as naturally as possible. (But don't try to prove it!)

Subsection 1.3 Sets and Functions

Recall that \(A\) is a subset of \(B\) (written \(A\subseteq B\) or \(A\subset B\)) if every element \(x\in A\) implies \(x\in B\text{.}\) That is, every element of \(A\) is also an element of \(B\text{.}\) When a set is defined as a subset of another set with a given property, we often use âset builderâ notation, such as

\begin{equation*} \{n\in \mathbb{Z} \mid 2\text{ divides } n\} \end{equation*}

for defining the even numbers. If a set has no elements, such as

\begin{equation*} \{P\text{ is a president of the USA before 2018 } \mid P\text{ is female } \}\text{,} \end{equation*}

we call it the empty set (\(\empset\) or \(\emptyset\)). The familiar intersection (\(\cap\)) and union (\(\cup\)) operators are very useful. We distinguish between \(\subseteq\) and \(\subsetneq\text{,}\) where in the latter the sets are not equal. Finally, we define \(A\setminus B\) to be the set of all elements in \(A\) which are not in \(B\text{.}\)

Exercise 1.6.

Prove that \(\mathbb{Z}\subseteq \mathbb{Q}\text{.}\)

Exercise 1.7.

Decide whether \(\mathbb{N}\cup \{-n\,|\,n\in \mathbb{N}\}=\mathbb{Z}\text{.}\)

Exercise 1.8.

Find three nonempty sets \(A,B,C\) such that \(A\cap B=\empset\text{,}\) \(B\in C\text{,}\) and \(A\notin C\text{.}\)

Exercise 1.9.

Give an example where \(A\neq B\) but \(A\setminus B=\empset\text{.}\)

More interestingly, we can define infinite combinations of these things, if we have a collection of sets \(X_\alpha\text{,}\) one for each element \(\alpha\in A\) an infinite set.

Problem 1.10.

Find (nonempty) sets \(S_i\subseteq \mathbb{N}\) indexed by \(i\in \mathbb{N}\) such that \(S_{i+1}\subsetneq S_i\) and

\begin{equation*} \bigcap_{i=1}^{\infty} S_i=\empset\,\text{.} \end{equation*}
Problem 1.11.

Find (nonempty) sets \(S_i\subseteq \mathbb{N}\) indexed by \(i\in \mathbb{N}\) such that \(S_i\subsetneq S_{i+1}\) but

\begin{equation*} \bigcup_{i=1}^\infty S_i\neq \mathbb{N} \,\text{.} \end{equation*}

We should also recall that, for us, a function is a set.

Definition 1.12.

A function from a set \(A\) to a set \(B\) is a subset of the Cartesian product \(A\times B\) such that for each \(a\in A\) there exists a unique \(b\in B\) such that \((a,b)\in f\text{.}\) We call \(A\) the domain of \(f\text{,}\) and say that \(f(a)=b\) if \((a,b)\in f\text{.}\) We call \(B\text{,}\) the codomain of \(f\text{.}\)

Exercise 1.13.

Write some function from \(\mathbb{Z}\) to itself as a set of ordered pairs.

Definition 1.14.

Recall a few properties functions might have:

  • A function from \(A\) to \(B\) is onto if, for each \(b\in B\text{,}\) there is an \(a\in A\) such that \(f(a)=b\text{.}\)

  • A function from \(A\) to \(B\) is one-to-one if for each \(b\in B\) there exists at most one \(a\in A\) such that \(f(a)=b\text{.}\)

  • A function from \(A\) to \(B\) is a bijection if it is one-to-one and onto.

A function that is onto is sometimes called a surjection or said to be surjective, while a one-to-one function is sometimes called an injection or said to be injective. Another name for a bijection is a one-to-one correspondence. These notes will generally use the terms given in DefinitionÂ 1.14, but it is helpful to be familiar with multiple ways to express the same idea.

Problem 1.15.

Let \(f\colon A\to B\text{,}\) where \(A\) and \(B\) are sets. Prove that \(f\) is one-to-one if and only if \(f(a_1)=f(a_2)\) implies \(a_1 = a_2\text{.}\)

Exercise 1.16.

Suppose that \(f(x)=x^2\text{,}\) where \(A=B=\mathbb{Z}\text{.}\) Change \(A\) or \(B\) so that it is one-to-one. Find a way to change it to be onto.

Exercise 1.17.

Find a bijection from \(\mathbb{Z}\) to itself that is not the identity. If you remember what an inverse function is, find that, too.

There are two more important sets relevant to a given function.

Definition 1.18.

Let \(f:A\to B\) be a function.

  • If \(S\subseteq A\text{,}\) we call the image of \(S\) under \(f\text{,}\) denoted \(f(S)\text{,}\) the set

    \begin{equation*} \{f(x) \mid x\in S\}\,\text{.} \end{equation*}

  • If \(T\subseteq B\text{,}\) we call the preimage of \(T\) under \(f\text{,}\) denoted \(f^{-1}(T)\text{,}\) the set

    \begin{equation*} \{x\in A \mid f(x)\in T\}\,\text{.} \end{equation*}

Notice that the preimage exists whether or not the inverse function exists (and the notation \(f^{-1}\) refers only to the preimage in this course). We call the image of the entire domain the image of the function, \(f(A)\text{.}\)

Exercise 1.19.

If \(f(x)=x^2\) from \(\mathbb{Z}\) to itself, find \(f(\{-2,-1,0,1,2\})\) and \(f^{-1}(\{0,1,4\})\text{.}\)

Problem 1.20.

Find functions \(f\) and \(g\) and sets \(S\) and \(T\) such that \(f(f^{-1}(T))\neq T\) and \(g^{-1}(g(S))\neq S\text{.}\) (Note that this is not composition!)

Problem 1.21.

With all sets appropriately defined, only one of two following statements is necessarily true. Prove one and disprove the other.

\begin{equation*} f(X\cap Y)=f(X)\cap f(Y) \end{equation*}
\begin{equation*} f^{-1}(X\cap Y)=f^{-1}(X)\cap f^{-1}(Y) \end{equation*}

Can you add an assumption about \(f\) that makes the false statement true? If so, prove that statement under the additional assumption.

Subsection 1.4 Order Properties of the Real Numbers

We are almost ready to start some actual analysis, where the functions and sets will be defined on the real numbers. (As mentioned earlier, we will tacitly assume that numbers in this course are real numbers, unless we state otherwise.) In order to do this, we need some facts about real numbers, including the following Order Axioms previously promised.

The Order Axioms.

For all \(a,b,c\in \mathbb{R}\text{:}\)

  1. If \(a\lt b\) and \(b\lt c\text{,}\) then \(a\lt c\text{.}\)

  2. If \(a\neq b\text{,}\) then exactly one of \(a\lt b\) or \(b\lt a\) is true.

  3. If \(a\lt b\text{,}\) then \(a+c\lt b+c\text{.}\)

  4. If \(a\lt b\) and \(c>0\text{,}\) then \(ac\lt bc\text{;}\) if \(c\lt 0\text{,}\) then \(ac>bc\text{.}\)

  5. There exists an integer \(n\) such that \(n\leq a\lt n+1\text{.}\)

Exercise 1.22.

Why are \(n\) and \(n+1\) are the smallest possible such integers?

Definition 1.23.

Given these axioms, we say that the real numbers \(\mathbb{R}\) are linearly ordered because âless thanâ and âgreater thanâ have meaning.

The last axiom is sometimes called âArchimedeanâ. We call numbers greater than zero positive, and those greater than zero or equal to zero nonnegative; there is a similar definition for negative and nonpositive.

Adding the order axioms to the most basic facts about addition, subtraction, and multiplication already can allow us to prove slightly more advanced facts.

Under these axioms, the things you expect about the relationship between positive, negative, addition, and multiplication are true â such as âa positive times a negative is negativeâ and âa positive plus a positive is positiveâ.

I'll let multiple people present ProblemÂ 1.24 if they prove substantially different facts. You can use \(-a=-1\cdot a\) and \(a-b=a+(-b)\) as definitions to deal with the two possible subtraction facts.

Problem 1.24.

Prove at least three of these types of results.

Here is another useful fact related to the axioms. It may seem just as expected, but is quite different in style.

Theorem 1.25.

For every positive \(a\in\mathbb{R}\text{,}\) there is an integer \(N\) such that \(0\lt \frac{1}{N}\lt a\text{.}\)

Finally, there is a special function which should be introduced now.

Definition 1.26.

Given \(a\in\mathbb{R}\text{,}\) we define the absolute value of \(a\), denoted \(|a|\text{,}\) to be \(a\) if \(a\geq 0\) and \(-a\) otherwise.

Exercise 1.27.

Show that \(|-4|\) is what you expect, using this definition.

Exercise 1.28.

Show that \(|a|\geq 0\text{,}\) with equality only if \(a=0\text{.}\)

Exercise 1.29.

Show that \(|a|^2=a^2\text{.}\)

The next two statements are pretty useful, and best proved (for now) by cases, though they have more clever proofs too.

Theorem 1.30.

For all \(a,b\in\R\text{,}\) \(|ab|=|a||b|\text{.}\)

Theorem 1.31. Triangle Inequality.

Show that \(|a+b|\leq |a|+|b|\text{.}\)

The related theorem below is sometimes called the reverse triangle inequality.

Theorem 1.32.

Show that \(|a-b|\geq |\, |a|-|b| \,|\text{.}\)

To prove the following, be sure to review properties of the integers we assume without proofÂ 1.1.

Theorem 1.33.

Assume that there is a positive element of the preimage of \(\{2\}\) under the function \(f(x) = x^2\) from the reals to the reals; that is, assume \(\sqrt{2}\) exists. Show \(1\lt \sqrt{2}\lt 2\text{.}\)

In MATH 210, we saw that \(\sqrt{2}\) is not a rational number, so now we have an irrational between two rationals, by TheoremÂ 1.33. The following theorems are a good challenge to generalize this.

Theorem 1.34.

For all \(a,b\in\R\) with \(a \lt b\text{,}\) there exists \(q\in\Q\) such that \(a \lt q\lt b\text{.}\)

Theorem 1.35.

For all \(a,b\in\R\) with \(a \lt b\text{,}\) there exists \(r\in\irrat\) such that \(a \lt r\lt b\text{.}\)
