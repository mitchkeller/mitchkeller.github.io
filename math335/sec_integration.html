Section 5 Integration

Subsection 5.1 Definitions

Unlike with differentiation, we will need a number of auxiliary definitions for beginning integration.

Definition 5.1.

A set of points \(\{t_0,t_1,\ldots,t_n\}\) such that \(a=t_0\text{,}\) \(b=t_n\text{,}\) and \(t_{i-1}\lt t_{i}\) for all \(i\) is called a partition of the closed interval \([a,b]\text{.}\)

Exercise 5.2.

Give some partitions of \([2,4]\) and \([-1,0]\text{.}\) What would the points of a partition of \([0,1]\) be if \(t_{i}-t_{i-1}\) is the same for all \(i\text{?}\)

Definition 5.3.

Given \([a,b]\) and calling \(\frac{b-a}{n}=\Delta t\text{,}\) we may for convenience give the partition \(\left\{a,a+\Delta t, a+2\Delta t, \ldots ,b-\Delta t, b\right\}\) the name regular partition \(P_n\) (where \([a,b]\) is implicit).

Definition 5.4.

Given a partition \(P=\{t_0,t_1,\ldots,t_n\}\) of \([a,b]\text{,}\) let \(y_i\) be the supremum of \(\{f(x)\mid x\in [t_{i-1},t_i]\}\text{.}\) We say that the sum

\begin{equation*} U_{P}(f) = \sum_{i=1}^n y_i(t_i-t_{i-1}) \end{equation*}

is the upper sum for \(P\) of \(f\) on \([a,b]\text{.}\)

Definition 5.5.

Given a partition \(P=\{t_0,t_1,\ldots,t_n\}\) of \([a,b]\text{,}\) let \(z_i\) be the infimum of \(\{f(x)\mid x\in [t_{i-1},t_i]\}\text{.}\) We say the sum

\begin{equation*} L_{P}(f) = \sum_{i=1}^n z_i(t_i-t_{i-1}) \end{equation*}

is the lower sum for \(P\) of \(f\) on \([a,b]\text{.}\)

Definition 5.6.

A bounded function is a function with a bounded image set.

We will assume all functions in this section are bounded.

Exercise 5.7.

Give some upper and lower sums on some of these partitions with non-constant bounded functions.

Theorem 5.8.

Let \(f(x)=0\) for each number \(x\) in \([0,1]\) except \(x=0\text{,}\) and let \(f(0)=1\text{.}\) For all partitions of \([0,1]\text{,}\) we have \(U_{P}(f)>0\text{,}\) but for any positive number \(\varepsilon\) there is a partition \(P_\epsilon\) such that \(U_{P_\varepsilon}(f)\lt \varepsilon\text{.}\)

Problem 5.9.

Fully describe all lower sums of the function \(f\) from Theorem 5.8 on \([0,1]\text{.}\)

Problem 5.10.

Let \(f(x)=x\text{,}\) and let \(P_n\) be the (regular) partition of \([0,1]\) given by \(\left\{0,\frac{1}{n},\frac{2}{n},\ldots,\frac{n-1}{n},1\right\}\text{.}\) Compute \(U_{P_5}(f)\) and \(L_{P_5}(f)\text{,}\) and then give a formula for \(U_{P_n}(f)\) and \(L_{P_n}(f)\text{.}\)

Theorem 5.11.

Suppose that \(f\) is bounded. If \(m\) is a lower bound and \(M\) is an upper bound of \(f([a,b])\text{,}\) then for any partition \(P\) of \([a,b]\text{,}\) \(U_P(f)\leq M(b-a)\) and \(L_P(f)\geq m(b-a)\text{.}\)

Theorem 5.12.

If \(f\) is bounded on \([a,b]\) and \(P\) is a partition of \([a,b]\text{,}\) then \(L_P(f)\leq U_P(f)\text{.}\)

Note that as a result of the previous problems that the set of all upper and lower sums of \(f\) is a bounded point set.

Definition 5.13.

We say that a partition \(Q\) refines (or is a refinement of) a partition \(P\) if \(P\) and \(Q\) are both partitions of \([a,b]\) and \(P\subseteq Q\text{.}\)

Exercise 5.14.

Take some partitions from previous examples, and refine them.

Theorem 5.15.

If \(P\) and \(P'\) are both partitions of \([a,b]\text{,}\) then there is a partition \(Q\) which is a refinement of both.

Theorem 5.16.

If \(f\) is a bounded function on \([a,b]\) and \(P\) and \(Q\) are partitions of \([a,b]\) such that \(Q\) refines \(P\text{,}\) then \(L_P(f)\leq L_Q(f)\) and \(U_P(f)\geq U_Q(f)\text{.}\)

We are nearly ready to define (definite) integrals.

Definition 5.17.

With \(f\) a bounded function on \([a,b]\text{,}\) we define the upper integral from \(a\) to \(b\) of \(f\) to be the infimum of the set of all upper sums for \(f\) on \([a,b]\text{.}\) We denote the upper integral by \(\overline{\int_a^b f}\text{.}\)

Definition 5.18.

With \(f\) a bounded function on \([a,b]\text{,}\) we define the lower integral from \(a\) to \(b\) of \(f\) to be the supremum of the set of all lower sums for \(f\) on \([a,b]\text{.}\) We denote the lower integral by \(\underline{\int_a^b f}\text{.}\)

Exercise 5.19.

Why do the upper integral and lower integral always exist in this context?

Theorem 5.20.

If \(f\) is bounded on \([a,b]\text{,}\) then \(\overline{\int_a^b f}\geq \underline{\int_a^b f}\text{.}\)

Theorem 5.21.

If we let \(f(x)\) be the function such that \(f(x)=1\) if \(x\in\mathbb{Q}\) but \(f(x)=0\) otherwise, then \(\overline{\int_0^1 f}> \underline{\int_0^1 f}\text{.}\)

We can get more interesting conclusions if \(f\) has more properties.

Theorem 5.22.

If \(f\) is continuous on \([a,b]\text{,}\) \(f(x)\geq 0\) for all \(x\in [a,b]\text{,}\) and for some \(z\in [a,b]\) we have that \(f(z)>0\text{,}\) then \(\overline{\int_a^b f}>0\text{.}\)

Definition 5.23.

For \(f\) a bounded function on a closed interval \([a,b]\text{,}\) if \(\overline{\int_a^b f}=\underline{\int_a^b f}\text{,}\) then we say that \(f\) is integrable on \([a,b]\) and call this common value the integral of \(f\) from \(a\) to \(b\text{.}\) We denote the integral, if it exists, by

\begin{equation*} \int_a^b f\text{ or } \int_a^b f(x)\, dx\;\text{.} \end{equation*}

Traditionally when this value exists we say that \(f\) is Riemann integrable and the number \(\int_a^b f\) is the Riemann integral. Technically, however, it is the Darboux integral, with Riemann integrals coming from so-called Riemann sums. The two notions can be proved equivalent.

Subsection 5.2 Integrable Functions

Since we saw above that the lower and upper integrals can be distinct, there exist nonintegrable bounded functions (in this sense). Luckily, there also exist integrable functions.

Exercise 5.24.

Show that constant functions \(f(x)=c\) are integrable over any closed interval.

The information we have from previous examples is not quite enough to show that we have more integrable functions, even though you are familiar with computing integrals. Since many applications in a variety of fields come directly from approximations involving the sums we have been computing, it would be best to have exact values for these applications, not just approximations from the sums. Keep this in mind as we progress.

A first step in this direction is the following characterization, which is very useful in showing functions are integrable.

Theorem 5.25.

Let \(f\) be a bounded function on \([a,b]\text{.}\) The function \(f\) is integrable if and only if for each positive number \(\epsilon\) there is a partition \(P\) of \([a,b]\) such that \(U_P(f)-L_P(f)\lt \epsilon\text{.}\)

Problem 5.26.

Pick one of these functions, show it is integrable on \([0,1]\text{,}\) and compute it. (Both are; you may want to use previous results.)

  • The function \(f\) such that \(f(x)=0\) for each number \(x\) in \([0,1]\) except \(x=0\text{,}\) where \(f(0)=1\text{.}\)

  • The function \(f(x)=x\text{.}\)

Exercise 5.27.

Let \(g(x)\) be the function such that \(f(x)=1/q\) if \(x=p/q\in\mathbb{Q}\) in lowest terms, but \(f(x)=0\) otherwise. Show this function is integrable.

This is not a lot of material to integrate. The next sets of theorems will vastly expand our repertoire, though not yet our formulas.

Definition 5.28.

A function is nondecreasing if for each \(x\lt y\) in the domain of \(f\) we have \(f(x)\leq f(y)\text{.}\) (We say that it is (strictly) increasing if \(f(x)\lt f(y)\) always. The concept of nonincreasing is defined with \(\geq\) in place of \(\leq\text{.}\))

Exercise 5.29.

Give several nondecreasing functions we haven't already used. (This is very similar to the equivalent sequence concept!)

Theorem 5.30.

If \(f\) is a nondecreasing bounded function on a closed interval \([a,b]\text{,}\) then \(f\) is integrable.

You may want to write down a number of upper and lower sums to prepare for the previous problem. What previous result will likely be easiest to use to prove integration theorems?

Exercise 5.31.

It turns out that a nonincreasing bounded function \(f\) on a closed interval \([a,b]\) is integrable as well.

Exercise 5.32.

Since \(\sqrt{1-x^2}\) is nonincreasing on \([0,1]\text{,}\) this shows that

\(4\int_0^1 \sqrt{1-x^2}\,dx\) exists. What number is this?

The preceding results mean that most standard functions are integrable on certain intervals, even if we can't finish computing them yet; the next theorems verify that we can do it on most intervals, too.

Theorem 5.33.

Let \(c\in \mathbb{R}\text{.}\) If \(f\) is integrable on \([a,b]\text{,}\) then \(cf\) is integrable on \([a,b]\) and \(\int_a^b cf = c\int_a^b f\text{.}\)

Theorem 5.34.

If \(f\) and \(g\) are integrable on \([a,b]\text{,}\) then \(f+g\) is integrable and \(\int_a^b (f+g) = \int_a^b f +\int_a^b g\text{.}\)

Subsection 5.3 FTC

We will now head straight for the punchline to the story of calculus, the “fundamental theorems”.

We begin by proving some facts about the upper integral (they are also true for the lower integral). Remember, all functions are bounded.

Theorem 5.35.

If \([a,b]\) is a closed interval and \(c\in (a,b)\text{,}\) then \(\overline{\int_a^c f} +\overline{\int_c^b f} = \overline{\int_a^b f}\text{.}\)

On a related note, for upper (and indeed for lower and Riemann) integrals, we define \(\overline{\int_b^a f}=-\overline{\int_a^b f}\) and \(\overline{\int_a^a f} =0\text{.}\) Can you see why we do this?

Theorem 5.36.

If \(f\leq g\) on the interval \([a,b]\text{,}\) then \(\overline{\int_a^b f} \leq \overline{\int_a^b g}\text{.}\)

The next property is of course also true for the regular integral (when it exists), and is called the integral mean value theorem. Can you see why?

Theorem 5.37.

If \(f\) be a continuous function on \([a,b]\text{,}\) then there is a number \(c\in [a,b]\) such that \(\overline{\int_a^b f} = f(c)(b-a)\text{.}\)

The next thing we do is really expand our repertoire of integrable functions!

Theorem 5.38.

Let \(U\) be defined such that for each \(x\in [a,b]\text{,}\) \(U(x) = \overline{\int_a^x f}\text{,}\) and let \(L\) be defined likewise but with \(\underline{\int_a^x f}\text{.}\) If \(f\) is continuous on \([a,b]\text{,}\) then \(U\) and \(L\) are differentiable, with \(U'(x)=f(x)=L'(x)\text{.}\)

Problem 5.39.

In the previous theorem, show that in fact \(U(x)=L(x)\text{,}\) and conclude that if \(f\) is continuous on \([a,b]\) then \(f\) is integrable on \([a,b]\text{.}\)

And finally we get the most beloved theorems of all. You should be able to prove these without lots of subtle work, using the facts about lower and upper integrals.

Theorem 5.40. Fundamental Theorem of Calculus Part I.

Let \(f\) be a continuous function on \([a,b]\text{.}\) If \(F\) is defined such that for each \(x\in [a,b]\text{,}\) \(F(x) = \int_a^x f\text{,}\) then for each point \(c\in [a,b]\text{,}\) \(F\) has a derivative at \(c\) and we can compute \(F'(c)=f(c)\text{.}\)

Theorem 5.41. Fundamental Theorem of Calculus Part II.

If \(g\) is a function on \([a,b]\text{,}\) \(g\) has a derivative at each point of \([a,b]\text{,}\) and the function \(g'\) is itself continuous at each point in \([a,b]\text{,}\) then \(\int_a^b g' = g(b)-g(a)\text{.}\)

These are the keys which made the efforts of Newton and Leibniz into the calculus—something one could calculate with. It is worth noting that this final theorem is true even if \(f'\) is just integrable—truly a great theorem.
