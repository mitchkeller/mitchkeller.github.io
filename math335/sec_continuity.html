Section 3 Continuity

Definition 3.1.

We say that a function \(f\) is continuous at a point \(x\) in its domain (or at the point \((x,f(x))\)) if, for any open interval \(S\) containing \(f(x)\text{,}\) there is an open interval \(T\) containing \(x\) such that if \(t\in T\) is in the domain of \(f\text{,}\) then \(f(t)\in S\text{.}\)

Definition 3.2.

A function \(f\) is continuous if it is continuous at every point in its domain.

Let's show that this definition of continuity behaves the way we expect from calculus. The first example will make it particular easy to find \(T\text{,}\) given \(S\text{.}\)

Example 3.3.

Show that \(f(x)=x\) is continuous for all \(x\in \mathbb{R}\text{.}\)

Example 3.4.

Pick one: Show that either \(f(x)=2x\) or \(f(x)=x+3\) is continuous for all \(x\in \mathbb{R}\text{.}\)

Problem 3.5.

Show that any linear function is continuous for all \(x\in \mathbb{R}\text{.}\)

Problem 3.6.

Let \(f(x)=1\) if \(x\in[0,1]\text{,}\) and \(f(x)=0\) otherwise. Determine and prove exactly for which \(x\) this function is continuous.

Problem 3.7.

Let \(f(x)=x^2\) (on \(\mathbb{R}\)). Show that \(f\) is continuous at \(x=2\text{.}\) (You may assume that square roots exist.)

Problem 3.8.

Define a function \(g:\{0\}\to \mathbb{R}\) by \(g(0)=0\text{.}\) Show that \(g\) is continuous at \(x=0\text{.}\)

Problem 3.9.

Let \(f(x)=1\) if \(x\in \mathbb{Q}\text{,}\) and \(f(x)=0\) otherwise. Find all points \(x\) where \(f\) is continuous.

It turns out that we can connect continuity to sequences, too.

Definition 3.10.

We say that a function \(f\) is sequentially continuous at a point \(x\) if, for every sequence \(\{x_i\}_{i=1}^\infty\) (in the domain of \(f\)) converging to \(x\text{,}\) it is also true that \(\{f(x_i)\}_{i=1}^\infty\) converges to \(f(x)\text{.}\)

Problem 3.11.

If a function \(f\) is continuous at \(x\text{,}\) it is also sequentially continuous at \(x\text{.}\)

Problem 3.12.

If \(f\) is sequentially continuous at \(x\text{,}\) it is continuous at \(x\text{.}\)

Proof.

Definitely by contradiction, create sequence explicit that is never inside of \(S\text{.}\)

So these concepts are equivalent on the real numbers. However, we still distinguish between them, since this isn't the case for all sets. See, for instance, http://www.youtube.com/watch?v=sZ5fBHGYurg.

The sequential way of thinking of continuity makes proving some basic facts easier. Be clear on the logic in the next two, and don't forget what we've already proved or assumed about sequence convergence.

Problem 3.13.

If \(c\in\mathbb{R}\) is a constant and if \(f\) and \(g\) are continuous at a point \(x\text{,}\) then so are the functions \(cf\) and \(f+g\text{.}\)

Proof.

This could be done directly or via the sequential continuity and sequence theorems.

Problem 3.14.

If \(f\) and \(g\) are continuous at a point \(x\text{,}\) so is the function \(fg\text{.}\)

Proof.

In principle, you really need to connect to sequential ones because I say that we have this property for sequences.

Example 3.15.

All polynomials in \(x\) are continuous for all \(x\) (here, in the full domain \(\mathbb{R}\))

Before finally moving to calculus proper, we'll deal briefly with two foundational theorems about \(\mathbb{R}\) and continuous functions that have everything to do with completeness.

  • The Extreme Value Theorem:.

    Let \(I\) be a closed interval and suppose that \(f\) is a function which is continuous on \(I\text{.}\) Then there exists \(x_M\in I\) such that \(f(x_M)\geq f(x)\) for all \(x\in I\text{.}\)

  • The Intermediate Value Theorem:.

    Suppose \(f\) is a continuous function on a closed interval \(I=[a,b]\text{,}\) and that \(f(a)\lt 0\lt f(b)\text{.}\) There there is a point \(x\in [a,b]\) such that \(f(x)=0\text{.}\)

Problem 3.16.

Suppose \(f\) is a continuous function on a closed interval \(I=[a,b]\) and that \(f(a)\lt c\lt f(b)\text{.}\) Prove there is a point \(x\in [a,b]\) such that \(f(x)=c\text{.}\)

Note that this is also true if \(f(b)\lt c\lt f(a)\text{.}\)

One way to get a handle on these theorems is to find counterexamples when the hypotheses are not satisfied.

Problem 3.17.

Find a continuous function on \(\mathbb{R}\) which does not satisfy the conclusions of the Extreme Value Theorem; do the same for a discontinuous function on \([0,1]\text{.}\)

Proof.

Need to show that in fact continuous or not, but I don't get quite as picky here.

Problem 3.18.

Find a continuous function on \([0,1]\cup [2,3]\) with \(f(0)=-1,f(3)=1\) which does not satisfy the conclusions of the Intermediate Value Theorem; do the same for a discontinuous function on \([0,1]\) where \(f(0)=-1,f(1)=1\text{.}\)

Proof.

Need to show that in fact continuous or not, but I don't get quite as picky here.

Proving the EVT is most often done using the sequential definition of continuity (or other definitions); it's rather tricky to do it using only the supremum definition. Here is one piece of the proof (this piece actually applies to non-continuous functions).

Problem 3.19.

Let \(f\) be a function on \([0,1]\text{,}\) and assume that the image \(f([0,1])\) has a supremum. Show (without using the EVT) there is a sequence of points \(\{x_i\}_{i=1}^\infty\) in \([0,1]\) such that \(\{f(x_i)\}_{i=1}^\infty\) converges to that supremum.

Example 3.20.

Explain why that's not enough, by itself, to prove the whole Extreme Value Theorem.

Proving the IVT, on the other hand, can definitely be done using just the set definitions of completeness and continuity. (One can even formalize that the EVT is ‘harder’ to prove, requiring both the axioms needed for the IVT and something called the ‘Weak König Lemma’.) The next problem should get your creative juices flowing as to how to find the \(x\) such that the IVT holds for that \(x\text{.}\) Be careful to show that your \(x\) exists!

Problem 3.21.

Let \(f\) be any function on \([0,1]\) such that \(f(0)=-1\text{,}\) \(f(1)=1\text{,}\) and \(f([0,1])=\{-1,1\}\text{.}\) Show (without using the IVT) that there is an \(x\in [0,1]\) such that \(f\) is not continuous at \(x\text{.}\)

Problem 3.22.

Prove the Intermediate Value Theorem.

Proof.

Make the set of \(\{x\in [a,b]\mid f(x)\leq 0\}\text{.}\)

Problem 3.23.

The image of a closed interval under a continuous function is either a closed interval or a single point.
