Section 3 Continuity

Subsection 3.1 Definition

Definition 3.1.

We say that a function \(f\) is continuous at a point \(x\) in its domain (or at the point \((x,f(x))\)) if, for any open interval \(S\) containing \(f(x)\text{,}\) there is an open interval \(T\) containing \(x\) such that if \(t\in T\) is in the domain of \(f\text{,}\) then \(f(t)\in S\text{.}\)

Definition 3.2.

A function \(f\) is continuous if it is continuous at every point in its domain.

Let's show that this definition of continuity behaves the way we expect from calculus. The first exercise will make it particular easy to find \(T\text{,}\) given \(S\text{.}\)

Exercise 3.3.

Show that \(f(x)=x\) is continuous for all \(x\in \mathbb{R}\text{.}\)

Exercise 3.4.

Pick one: Show that either \(f(x)=2x\) or \(f(x)=x+3\) is continuous for all \(x\in \mathbb{R}\text{.}\)

Theorem 3.5.

If \(f\) is a linear function, then \(f\) is continuous for all \(x\in \mathbb{R}\text{.}\)

Problem 3.6.

Let \(f(x)=1\) if \(x\in[0,1]\text{,}\) and \(f(x)=0\) otherwise. Determine and prove exactly for which \(x\) this function is continuous.

You may assume that square roots exist when proving the following theorem.

Theorem 3.7.

If \(f(x)=x^2\) (on \(\mathbb{R}\)), then \(f\) is continuous at \(x=2\text{.}\)

Theorem 3.8.

If \(g:\{0\}\to \mathbb{R}\) is defined by \(g(0)=0\text{,}\) then \(g\) is continuous at \(x=0\text{.}\)

Problem 3.9.

Let \(f(x)=1\) if \(x\in \mathbb{Q}\text{,}\) and \(f(x)=0\) otherwise. Find all points \(x\) where \(f\) is continuous.

Subsection 3.2 Connection to Sequences

It turns out that we can connect continuity to sequences, too.

Definition 3.10.

We say that a function \(f\) is sequentially continuous at a point \(x\) provided that for every sequence \(\{x_i\}_{i=1}^\infty\) (in the domain of \(f\)) converging to \(x\text{,}\) it is also true that \(\{f(x_i)\}_{i=1}^\infty\) converges to \(f(x)\text{.}\)

Theorem 3.11.

If a function \(f\) is continuous at \(x\text{,}\) it is also sequentially continuous at \(x\text{.}\)

Theorem 3.12.

If \(f\) is sequentially continuous at \(x\text{,}\) it is continuous at \(x\text{.}\)

So these concepts are equivalent on the real numbers. However, we still distinguish between them, since this isn't the case for all sets.

The sequential way of thinking of continuity makes proving some basic facts easier. Be clear on the logic in the next two, and don't forget what we've already proved or assumed about sequence convergence.

Theorem 3.13.

If \(c\in\mathbb{R}\) is a constant and if \(f\) and \(g\) are continuous at a point \(x\text{,}\) then so are the functions \(cf\) and \(f+g\text{.}\)

Theorem 3.14.

If \(f\) and \(g\) are continuous at a point \(x\text{,}\) so is the function \(fg\text{.}\)

Exercise 3.15.

All polynomials in \(x\) are continuous for all \(x\) (here, in the full domain \(\mathbb{R}\))
