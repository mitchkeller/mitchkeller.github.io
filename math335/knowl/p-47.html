Recall a few properties functions might have.

  • A function from \(A\) to \(B\) is onto if, for each \(b\in B\text{,}\) there is an \(a\in A\) such that \(f(a)=b\text{.}\)

  • A function from \(A\) to \(B\) is one-to-one if when \(a\neq a' \in A\text{,}\) then \(f(a)\neq f(a')\in B\text{.}\)

  • A function from \(A\) to \(B\) is a bijection if it is one-to-one and onto.
