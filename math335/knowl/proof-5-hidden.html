One easy way to do this is to use the next problem, after noting that \(\sqrt{2}-1>0\) and that then one can divide by some natural number. But there is also a direct way. What was it again? Hmm, it may use a more robust version of Archimedean property wherein there is an integer \(m\) such that \(m(b-a)>\) your favorite positive irrational. How does one get that from our version of Archimedean? Luke C. has a nice one where \(b-a>1/n\) so \(nb-na>1\) so \(2nb-2na>2\text{,}\) so — again, one needs to know something about how far apart things have to be — \(2nb>2na+\sqrt{2}>2na>0\) which suffices.
