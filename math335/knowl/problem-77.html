Problem 5.25.

Pick one of these functions, show it is integrable on \([0,1]\text{,}\) and compute it. (Both are; you may want to use previous problems.)

  • The function \(f\) such that \(f(x)=0\) for each number \(x\) in \([0,1]\) except \(x=0\text{,}\) where \(f(0)=1\text{;}\)

  • The function \(f(x)=x\text{.}\)
