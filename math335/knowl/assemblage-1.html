The Order Axioms.

For all \(a,b,c\in \mathbb{R}\text{:}\)

  1. If \(a\lt b\) and \(b\lt c\text{,}\) then \(a\lt c\text{.}\)

  2. If \(a\neq b\text{,}\) then exactly one of \(a\lt b\) or \(b\lt a\) is true.

  3. If \(a\lt b\text{,}\) then \(a+c\lt b+c\text{.}\)

  4. If \(a\lt b\) and \(c>0\text{,}\) then \(ac\lt bc\text{;}\) if \(c\lt 0\text{,}\) then \(ac>bc\text{.}\)

  5. There exists an integer \(n\) such that \(n\leq a\lt n+1\text{.}\)
