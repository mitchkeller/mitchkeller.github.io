There are a few other tricky logic issues that we should definitely practice. First, or is a word that means many things. In most of math (and here), “A or B” means “A or B or both”. This is important in statements like

\begin{equation*} \text{ If } x\text{ or } y\text{ is even, then } xy\text{ is also even } \end{equation*}

since even if \(x\) and \(y\) were both even, \(xy\) would still be even.
