Proof

If \(x^2\) is an even integer, then so is \(x\text{.}\) If \(x\) is an odd integer, then so is \(x^2\text{.}\) (Or, not even.) If \(x\) is odd, then \(x=2k+1\) for some integer \(k\text{,}\) by definition of odd. Then \(x^2=(2k+1)^2=4k^2+4k+1\text{.}\) We can rewrite this as \(2(2k^2+2k)+1\text{,}\) and by the usual closure properties of integers \(x^2=2\ell +1\text{,}\) where \(\ell\) is the integer \(2k^2+2k\text{.}\) So \(x^2\) is also odd.
