Definition 1.13.

A function from a set \(A\) to a set \(B\) is a subset of the Cartesian product \(A\times B\) such that there is precisely one \((a,b)\in f\) for each \(a\in A\text{.}\) We call \(A\) the domain, and say that \(f(a)=b\) if \((a,b)\in f\text{.}\) There is no standard name for \(B\text{,}\) but if you insist on having one, use codomain.
