Problem 3.24.

Find a function \(f\) on a closed interval \(I\) such that \(f(I)\) is also a closed interval, but for which \(f\) is not continuous for some \(x\in I\text{.}\) Find a continuous function \(g\) on a closed interval \(I=[a,b]\) such that \(g(a)=g(b)\) but \(g(I)\) is not (just) the single point \(g(a)\text{.}\) Finally, find a function \(h\) on a closed interval \(I\) such that \(h(I)\) is not a closed interval.
