In the first two examples, we have the highly unusual situation where both \(P\implies Q\) and \(Q\implies P\) are both true. We say that this is a biconditional statement and write \(P\iff Q\text{,}\) saying

\begin{equation*} P\text{ is true if and only if } Q\text{ is true }\text{,} \end{equation*}

or “\(P\) is necessary and sufficient for \(Q\)”, or some variation thereon.
