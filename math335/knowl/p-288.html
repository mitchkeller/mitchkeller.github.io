For \(f\) a bounded function on a closed interval \([a,b]\text{,}\) if \(\overline{\int_a^b f}=\underline{\int_a^b f}\text{,}\) then we say that \(f\) is integrable on \([a,b]\) and call this common value the integral of \(f\) from \(a\) to \(b\text{.}\) We denote the integral, if it exists, by

\begin{equation*} \int_a^b f\text{ or } \int_a^b f(x)\, dx\;\text{.} \end{equation*}
