Definition 1.18.

Let \(f:A\to B\) be a function.

  • If \(S\subseteq A\text{,}\) we call the image of \(S\) under \(f\text{,}\) denoted \(f(S)\text{,}\) the set

    \begin{equation*} \{f(x) \mid x\in S\}\,\text{.} \end{equation*}

  • If \(T\subseteq B\text{,}\) we call the preimage of \(T\) under \(f\text{,}\) denoted \(f^{-1}(T)\text{,}\) the set

    \begin{equation*} \{x\in A \mid f(x)\in T\}\,\text{.} \end{equation*}
