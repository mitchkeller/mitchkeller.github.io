Section 2 Point Sets and Sequences

Subsection 2.1 Point Sets

Definition 2.1.

A point is simply an element of the real numbers \(\mathbb{R}\text{.}\)

Definition 2.2.

A point set is a nonempty subset of the real numbers.

You might well wonder why we need extra terms for these concepts. The reasons are mainly historical. Not only do these ideas undergird much more abstract ideas of sets, but in addition, there is a tradition worth preserving of inquiry-based learning courses using this terminology.

One of the interesting things about choosing to look at the whole real number line is that we have two additional useful axioms. They seem nearly trivial, but are very important.

Axiom 2.3. Point Set Axiom 1.

If \(p\) is a point, then there is a point less than \(p\) and a point greater than \(p\text{.}\)

Axiom 2.4. Point Set Axiom 2.

If \(p\neq q\) are two points, there is a point between them.

Note that these are closely related to the last few problems — but we are starting from scratch here, and you should treat these are the primary axioms we need for now.

Exercise 2.5.

Explain why the Point Set Axioms above seem reasonable based on things you know about real (or rational) numbers.

Exercise 2.6.

If instead of \(\mathbb{R}\) we chose as our “background set” some subset \(C\) of \(\mathbb{R}\text{,}\) these axioms might not hold. We can see this with \(C=[0,1]\cup[2,3]\text{,}\) with \(p=0\) in the first case and \(p=1,q=2\) in the second.

Definition 2.7.

We say that a point set \(O\) is a (finite) open interval if there are points \(a\neq b\) such that \(O\) is the point set consisting of all points between \(a\) and \(b\text{.}\) That is,

\begin{equation*} O=\{x \,\mid \, a\lt x\lt b\}=(a,b)\,\text{.} \end{equation*}
Definition 2.8.

We say that a point set \(I\) is a (finite) closed interval if there are points \(a\neq b\) such that \(I\) is the point set consisting of \(a\text{,}\) \(b\text{,}\) and all points between \(a\) and \(b\text{.}\) That is,

\begin{equation*} I=\{x \,\mid \,a\leq x\leq b\}=[a,b]\,\text{.} \end{equation*}

In either case, \(a\) and \(b\) are called the endpoints of the interval.

Exercise 2.9.

Give some examples of point sets that are not intervals.

Definition 2.10.

If \(M\) is a point set, we say that \(p\) is an accumulation point of \(M\) if every open interval containing \(p\) also contains a point of \(M\) different from \(p\text{.}\)

To word this in set notation, \(p\) is an accumulation point of \(M\) if, for every open interval \(O\) with \(p\in O\text{,}\) there is \(x\in O\) such that \(x\in M\setminus\{p\}\text{.}\)

Theorem 2.11.

If \(M\) is an open interval and \(p\in M\text{,}\) then \(p\) is an accumulation point of \(M\text{.}\)

Theorem 2.12.

If \(M\) is a closed interval and \(p\notin M\text{,}\) then \(p\) is not an accumulation point of \(M\text{.}\)

Problem 2.13.

Determine whether the endpoints of an open interval \(M\) are accumulation points of \(M\text{.}\)

It is worth exploring exactly how many points it is possible or impossible for \(M\) to have. The next two problems are just a start in investigating that.

In the next problem, you may wish to first show that \(M\) must have at least one point!

Problem 2.14.

Show that if \(M\) is a point set having an accumulation point, then \(M\) contains (at least) two points. Determine whether \(M\) must contain at least 3 points.

Problem 2.15.

Consider \(\mathbb{Z}\) as a point set; show that it has no accumulation points.

Theorem 2.16.

Let \(H\) and \(K\) be point sets. If \(p\) is an accumulation point of \(H\cap K\text{,}\) then \(p\) is an accumulation point of both \(H\) and \(K\text{.}\)

Theorem 2.17.

Let \(H\) and \(K\) be point sets. If \(p\) is an accumulation point of \(H\cup K\text{,}\) then \(p\) is an accumulation point of \(H\) or \(p\) is an accumulation point of \(K\text{.}\)

Theorem 2.18.

If \(M\) is the set of all reciprocals of elements of \(\mathbb{N}\text{,}\) then zero is an accumulation point of \(M\text{.}\)

Subsection 2.2 Sequences

We will next start to turn to connecting the concepts of point sets to more familiar ones from calculus, beginning with sequences.

Definition 2.19.

A sequence is a function from \(\mathbb{N}\) to \(\mathbb{R}\text{.}\)

Sometimes these are called point sequences to distinguish them from other potential sequences.

Even though technically the sequence is a function \(a\text{,}\) for any given \(i\in \mathbb{N}\) we will write \(a_i=a(i)\text{.}\) Then we may write a sequence \(a(1),a(2),a(3),\cdots\) as \(a_1,a_2,a_3,\cdots\) — or, more formally, \(\{a_i\}_{i=1}^\infty\text{.}\) In order to keep the index visible (this is especially useful when we combine sequences), we will occasionally abuse notation slightly and write \(\{a_i\}_{i=1}^\infty\) when referring to the function or the image set \(\{a(i) \mid i\in \mathbb{N}\}\text{.}\)

Exercise 2.20.

Write down several sequences \(a\) you are familiar with. If possible, give a general formula for each \(a_i\) in terms of \(i\text{.}\)

Exercise 2.21.

Give an example where the image set of a sequence \(\{a_i\}_{i=1}^\infty\) is finite.

There is a deep connection between sequences and accumulation points, which the next few problems will elucidate. First, a definition—one you may have seen in calculus in a different form.

Definition 2.22.

We say that the (point) sequence \(p=\{p_i\}_{i=1}^\infty\) converges to the point \(x\) if, for every any open interval \(S\) containing \(x\text{,}\) there exists an \(N\in \mathbb{N}\) such that for all integers \(n\) with \(n\geq N\text{,}\) \(p_n\in S\text{.}\)

We simply say that \(p\) converges if there exists a point \(x\) to which the sequence converges. One could even write \(p\to x\text{,}\) informally. If a sequence does not converge to any point \(x\text{,}\) then we say it diverges.

The first problem about this should be used as a place to test ideas for how to prove convergence. Make sure you remember all the axioms you've learned and facts you've proved about real numbers — you may need them!

Theorem 2.23.

Define a sequence given by \(p_n = \frac{1}{n}\) (remember, \(n\in\mathbb{N}\) is part of the definition of a sequence). We have that \(p=\{p_i\}_{i=1}^\infty\) converges to \(0\text{.}\)

Exercise 2.24.

Before tackling any of the next few problems, try actually writing down the first 10 or 12 elements of each sequence.

Theorem 2.25.

The sequence given by \(p_n = 1-\frac{1}{n}\) converges to \(1\text{.}\)

Theorem 2.26.

The sequence with even terms \(p_{2k}=\frac{1}{2k-1}\) and odd terms \(p_{2k-1}=\frac{1}{2k}\) converges to \(0\text{.}\)

Problem 2.27.

Consider the sequence with odd terms \(p_{2k-1}=\frac{1}{2k-1}\) and even terms \(p_{2k}=1+\frac{1}{2k}\text{.}\) Determine whether \(p\) converges to \(0\text{.}\)

Theorem 2.28.

The sequence given by \(p_{2k} = \frac{1}{2k}\) and \(p_{2k-1}=-\frac{1}{2k-1}\) converges to \(0\text{.}\)

The following problem connects our two concepts. The most profound property of the real numbers is part of this connection, as we'll soon see.

Theorem 2.29.

If \(p\) converges to the point \(x\) and for each \(i\in \mathbb{N}\text{,}\) \(p_i\neq p_{i+1}\text{,}\) then \(x\) is an accumulation point of the image set of \(\{p_i\}_{i=1}^\infty\text{.}\)

Exercise 2.30.

Why do we need the restriction that \(p_i\neq p_{i+1}\text{?}\) Is this an absolutely necessary restriction for \(x\) to be an accumulation point of the image set?

Theorem 2.31.

The sequence \(\left\{\frac{1}{i}\right\}_{i=1}^\infty\) does not converge to a point other than zero.

Theorem 2.32.

If \(p\) converges to the point \(x\) and \(y\) is a point different from \(x\text{,}\) then \(p\) does not converge to \(y\text{.}\)

Basic familiar facts about sequence convergence are recalled next. In these proofs, you may have to think a little more explicitly about what the intervals around \(x\) look like in order to combine sequences. Try doing some examples with explicit numbers first, to get a sense of how to approach them.

Theorem 2.33.

If \(c\) is a number and \(p=\{p_i\}_{i=1}^\infty\) converges to \(x\text{,}\) then \(cp\) (which means what you think it does) converges to \(cx\text{.}\)

Theorem 2.34.

If \(q=\{q_i\}_{i=1}^\infty\) converges to \(y\) and \(p=\{p_i\}_{i=1}^\infty\) converges to \(x\text{,}\) then \(\{q_i+p_i\}_{i=1}^\infty\) (which means what you think it does) converges to \(y+x\text{.}\)

Products and quotients of sequences behave like you think they will as well, and you can use these facts in the rest of the notes. We will include one special case soon.

Subsection 2.3 Completeness

Now we introduce a few more definitions that will lead us to the key axiom for the real numbers—completeness. We'll continue to see interplay between sequences and sets.

Definition 2.35.

We say that a point set \(M\) is bounded if \(M\) is a subset of some closed interval.

Definition 2.36.

We say that a point set \(M\) is bounded above if there is a point \(z\) such that if \(x\in M\) then \(x\leq z\text{;}\) such a point is an upper bound.

Exercise 2.37.

The property of a point set \(M\) being bounded below and the notion of a lower bound are defined similarly. Define them carefully.

Exercise 2.38.

Show that a point set (in \(\mathbb{R}\)) being bounded is the same as it being bounded above and below.

Exercise 2.39.

Find all upper bounds for \((0,1)\text{,}\) \([0,1]\text{,}\) and \((0,1)\cap \mathbb{Q}^{C}\) (irrationals between 0 and 1).

In the next theorem, remember that we ‘abuse notation’ by using \(\{p_i\}_{i=1}^\infty\) to mean more than one mathematical object.

Theorem 2.40.

If the sequence \(p\) converges to the point \(x\text{,}\) then the image set \(\{p_i\}_{i=1}^\infty\) is bounded.

You can use this concept to prove some of the more difficult sequence convergence properties.

Theorem 2.41.

If \(q\) converges to \(0\) and \(p\) converges to \(x\text{,}\) then \(\{q_i\cdot p_i\}_{i=1}^\infty\) (which means what you think it does) converges to \(0\text{.}\)

Now we start to approach the heart of why calculus works.

Definition 2.42.

We say \(p\) is a least upper bound (or supremum) of \(M\) if \(p\) is an upper bound of \(M\) and \(p\leq q\) for any other upper bound \(q\) of \(M\text{.}\)

Exercise 2.43.

Define the greatest lower bound and infimum by analogy.

Exercise 2.44.

Find the suprema of \((0,1)\text{,}\) and \((0,1)\cap \mathbb{Q}^{C}\text{.}\) If we could apply the definition of supremum to \(\emptyset\text{,}\) what would its supremum be?

Theorem 2.45.

If a point set has a supremum, then that supremum is unique.

Problem 2.46.

If \(M\) and \(N\) are point sets with suprema, characterize the supremum of \(M\cup N\text{.}\)

If \(M\) and \(N\) are point sets, define \(cM\) to be \(\{cx \mid x\in M\}\) and \(M+N=\{x+y \mid x\in M, y\in N\}\text{.}\) For Theorem 2.47, I will take volunteers for the scalar multiple and sum portions separately.

Theorem 2.47.

Assuming \(M\) and \(N\) have suprema, \(\sup(cM)\) is \(c\sup(M)\) (given \(c>0\)) and \(\sup(M+N)=\sup(M)+\sup(N)\text{.}\)

Exercise 2.48.

Show that \(c\inf(M) = \sup(cM)\) if \(c\lt 0\text{.}\) What other properties are there relating \(\inf\text{,}\) \(\sup\text{,}\) and \(c\text{?}\)

The reason the supremum is so important is this fundamental axiom:

Axiom 2.49. Completeness of the Real Numbers.

If \(M\) is a point set and is bounded above, then \(M\) has a supremum.

Exercise 2.50.

Find a sequence \(p\) such that the image set is unbounded \(\{p_i\}_{i=1}^\infty\) and hence does not have a supremum.

Exercise 2.51.

Show Axiom 2.49 is not true if one requires that the supremum be a rational number.

The next problem is an equivalent “sequence definition” formulation of completeness of the real numbers.

Definition 2.52.

We say that a sequence \(p\) is nondecreasing if \(p_i\leq p_{i+1}\) for all \(i\in \mathbb{N}\text{.}\) (The concept of nonincreasing is defined similarly.)

Exercise 2.53.

Replace the \(\leq\) above with \(\lt\) to define the notion of (strictly) increasing. Find examples of nondecreasing sequences which are not increasing (and similarly for nonincreasing/decreasing).

Theorem 2.54.

If \(p\) is a nondecreasing sequence such that the image set \(\{p_i\}_{i=1}^\infty\) is bounded above, then \(p\) converges to some point \(x\text{.}\)

Problem 2.55.

Assume Theorem 2.54 is true and use it to prove the completeness axiom for point sets.

Why is all this so important? One reason is that we can use the completeness of the reals to prove one of the axioms we snuck in earlier. It may be thought of as the “real” reason why the following is true, sicne open intervals can be as small as we need them to be.

Problem 2.56.

Using the previous problem (but not its proof!), show that for any point \(x\text{,}\) there is an \(n\in\Z\) such that \(n\gt x\text{.}\)

A use of this alternative definition of completeness (that nondecreasing sequences bounded above converge) is to prove the following famous theorem, which we state but will not prove:

The Bolzano-Weierstrass Theorem.

Let us call a sequence \(\{b_k\}_{k=1}^\infty\) a subsetquence of another sequence \(\{a_n\}_{n=1}^\infty\) if there is a sequence of natural numbers \(\{n_i\}_{i=1}^\infty\) with \(n_i \lt n_{i+1}\) for all \(i\) such that \(b_k = a_{n_k}\text{.}\) Every sequence with bounded image set has a convergent subsequence.

Exercise 2.57.

Let \(p_n=(-1)^n\text{.}\) Find a convergent subsequence of this sequence guaranteed by the The Bolzano-Weierstrass Theorem.

There are also many famous properties of sequences which the sequential definition of completeness allows us to prove, but we will have to skip these in order to start tackling more familiar concepts from calculus, which we do next.
